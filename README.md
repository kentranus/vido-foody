# Vido Foody POS - Final Original UI

This package keeps the POS Flutter UI from the original `vido_pos_dual` app.

Use this package for the POS repo. It keeps the original POS layout and adds the Vido Foody logo, Online Orders, Kiosk, Reports, Bills, Settings entry points, PAX payment labels, and the Node.js backend APIs for kiosk/online orders/refund/settlement/receipt printing.

Upload the contents of this folder to the POS GitHub repo root:

- `.github`
- `vido-foody-backend`
- `vido-foody-flutter`
- `README.md`

The Flutter app is inside `vido-foody-flutter` so GitHub Actions can build the APK from the same repo structure.

Do not mix this with the older redesigned POS folders. Upload this package's contents to the GitHub repo root.
